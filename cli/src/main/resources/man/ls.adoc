LS(1)
===========
:doctype: manpage


NAME
----
*ls* - list directory contents


SYNOPSIS
--------
*less* ['OPTIONS'] 'FILE'


DESCRIPTION
-----------
List information about the FILEs (the current directory by default).


OPTIONS
-------

*-l*::
     Use a long listing format.
